# Privacy Policy: entra-access-graph-engine

## What I Collect

This tool reads Entra ID directory data (users, groups, roles, applications, service principals) via the Microsoft Graph API using read-only application permissions. No data is transmitted to any server other than the Microsoft Graph API endpoint and your configured export destination.

## Data Storage

All graph data is held in memory during the scan and written to the output file you specify. No data is stored persistently by this tool. Output files may contain directory object names and IDs from your Entra ID tenant.

## Telemetry

This tool does not include any built-in telemetry, crash reporting, or usage tracking.

## Contact

Open a GitHub issue for privacy-related questions.
